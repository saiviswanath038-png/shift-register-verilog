# Shift Register Design (Verilog)

This project implements a 4-bit shift register using Verilog HDL.

## Features
- Shift Left and Shift Right operations
- Parallel Load functionality
- Serial Input and Output
- Clock-based sequential design

## Tools Used
- Verilog HDL
- ModelSim

## Description
The design supports multiple modes such as hold, shifting, and parallel loading using control signals. The functionality is verified using a testbench.
